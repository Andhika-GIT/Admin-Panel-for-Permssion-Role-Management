## About App

Admin panel built with Laravel 8, Jetstream, Inertia, AdminLTE, Spatie's Laravel-permissions, Jetstrap, and Bootstrap 5 for Role and Permission Management

## How To Use

-   Clone or Download the source code
-   Run NPM INSTALL
